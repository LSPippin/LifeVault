# LifeVault

A personal information management web application built with Django and PostgreSQL.

LifeVault allows users to organize important life information — health records,
insurance, home maintenance, finances, pets, and more — in one centralized place.

## Tech Stack
- Python / Django
- PostgreSQL
- HTML / CSS

## How to Run Locally
1. Clone the repo
2. Create a virtual environment and install requirements
3. Set up your .env file with database credentials
4. Run migrations
5. Start the server with python manage.py runserver
